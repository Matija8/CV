\section{Projects}


\cvlistitem{\textbf{Mail client and server (June 2020.)} - Web programming course project. In a team with 4 colleagues worked on implementing a gmail clone. Technologies used on the client were Angular 9, RxJS and Angular Material. Node.js and postgres with TypeORM were used on the server. Team role was to synchronize message streams and do general client side programming(\href{https://github.com/Matija8/Post.ar}{link}).}

\cvlistitem{\textbf{Turn based video game (February 2020.)} - Software development course project. Written in C++, also using the Qt5 GUI library. Players move their units on the board, with the objective being to eliminate all their opponents units(\href{https://github.com/Matija8/Tactical-turn-based-game}{link}).}

\cvlistitem{\textbf{Minesweeper (January 2020.)} - Project for the Levi9 JavaScript course. Project consists of 3 parts: 1) Vanilla JS video game (minesweeper); 2) Node.js server + MongoDB for storing highscores; 3) React SPA leaderboard for displaying highscores(\href{https://github.com/Matija8/Js-Minesweeper}{link}).}

\cvlistitem{\textbf{Free throw shooting (May 2019.)} - Mathematical modeling course project. In a team with a colleague written a paper on calculating the optimal shot angle in relation to the height of the free throw shooter. Angle calculation was implemented using Python and the NumPy library(\href{https://github.com/Matija8/Slobodna-bacanja-Latex-}{link}).}

\cvlistitem{\textbf{Side-scroller video game (January 2018.)} - Computer graphics course project. Written in C, using OpenGL for graphics rendering. The goal of the game is to reach the last platform(\href{https://github.com/Matija8/Sidescrolling-platform-video-game}{link}).}
